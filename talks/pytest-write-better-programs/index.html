Pytest : Write better programs

Tapasweni Pathak

Pytest is a full featured testing tool that can make your testing experience better. This talk will introduce you to pytest and show how easy it is to get started with pytest in your own project.

Good practices are crucial to effective testing. It will also discuss how to write faster tests and how to improve test runs time by running tests in parallel.
